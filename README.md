# **antHurry-project**

## 💻 프로젝트 소개

- 급한 일이 있을 때 주변의 도움이 필요한 사람들
- 누군가에게 도움을 줘서 금전적 보상을 받고싶은 사람들 
- 채팅을 통해 서로 커뮤니케이션이 가능합니다.
- 상대방의 별점을 통해 신뢰를 확인할 수 있습니다.
